# Changelog

All notable changes to this project will be documented in this file.

## 1.3.5

- Added `--reverse` flag to swap front and back of cards
- Added `--dry-run` / `--preview` to preview cards without writing a file
- Added `--tags` option for comma-separated Anki tags
- Added `-i` / `--extra-input` to merge multiple input files
- Added CLI options and tests

## 1.0.0

- Initial release
- Basic and sound card models
- UTF-8 and chardet encoding detection
- Click-based CLI
